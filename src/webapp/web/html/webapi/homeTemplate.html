voidbase

Implementation of queue-based computing paradigm, suited for search and analytics processing in soft real-time environments.

Queue-based computing represents an approach to real-time distributed processing of large-scale data revolving around the notion of dynamic allocation of queues, registering appropriate convolutional operation to each queue and providing a set of high-order operations for providing desired data processing functionalities.

Features

  • queue-based processing framework
  • integrated data persistence engine
  • integrated feed processing engine
  • support for statistical analysis on top of real time data
  • implementation of fundamental online machine learning algorithms
  • integrated visualization engine
